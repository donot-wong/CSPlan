# CScan
CScan是一个半自动化扫描器，需要配合人工进行漏洞扫描，扫描器以乙方安全人员的视角进行设计和开发，
扫描器的目标是配合安全人员的能力对目标进行漏洞检测，实现目标漏洞检测准确率100%


### 架构设计
基础数据采集模块 - Chrome插件
数据处理模块 - 数据预处理及存储
任务调度模块 - 扫描及其他任务调度
扫描平台模块 - 只做扫描平台，不做扫描策略
扫描策略模块 - 安全人员的主要战场，插件化扫描策略配置


### 核心技术
Chrome插件
Python
Celery
Redis
RabitMQ
...
